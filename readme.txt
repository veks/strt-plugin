STRT — кастомные блоки и расширения редактора для Gutenberg и WooCommerce.

== Description ==

STRT — плагин, добавляющий пользовательские Gutenberg-блоки, улучшения редактора и интеграции с WooCommerce.
Позволяет быстро расширять функциональность сайта без вмешательства в код темы.

**Особенности:**
* Кастомные блоки для Gutenberg
* Поддержка динамических блоков с `render_callback`
* Поддержка шаблонов блоков и InnerBlocks
* Поддержка WooCommerce: отображение блоков в карточке товара, корзине и на чекауте

== Installation ==

1. Скачайте и распакуйте архив плагина в директорию `/wp-content/plugins/strt/`
2. Активируйте плагин через меню **Плагины** в админке WordPress
3. Используйте новые блоки в редакторе Gutenberg
